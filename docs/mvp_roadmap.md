# VitalLens AI - MVP Roadmap

## Phase 1: Core Intelligence (Weeks 1-4)
**Goal**: Establish the "Score + Explain" loop.
- [ ] Infrastructure setup (Kafka, Postgres, Redis).
- [ ] Auth & User Profile Services.
- [ ] Metric Ingestion (Mock Provider/CSV support).
- [ ] Base Sleep/Recovery/Strain computation logic.
- [ ] AI Service integration (Basic daily explanation narrative).
- [ ] Minimalist Dark UI Dashboard (Web).

## Phase 2: Behavioral Insights (Weeks 5-8)
**Goal**: Link habits to recovery outcomes.
- [ ] Journal Service (Logging habits/symptoms).
- [ ] Habit Impact Engine (Correlating journal entries with RSV/RHR).
- [ ] Weekly Trend reports.
- [ ] Proactive Alerts (Anomaly detection for 3+ day trends).
- [ ] AI Coach Chat (Interactive Q&A over data).

## Phase 3: Predictive & Premium (Weeks 9-12)
**Goal**: Forecasting and real-world integration.
- [ ] Predictive Recovery forecasting (Next-day risk).
- [ ] Real wearable API integrations (WHOOP, Apple Health).
- [ ] PDF Export for Medical/Coaching use.
- [ ] Lifestyle Simulator ("What if I sleep 8 hours?").
- [ ] Team/Coaching dashboard mode.

## Future Horizons
- [ ] Advanced Cardiovascular age estimation.
- [ ] Integration with nutrition and supplement tracking.
- [ ] Social/Community challenges based on recovery consistency.
